Để có thể setup project thì ta:
- Import file database.sql vào để tạo dữ liệu. 
- Lấy toàn bộ các thư mục trong source code bỏ vào htdocs
- Để truy cập trang chủ, thì ta gõ vào thanh địa chỉ: localhost:(port do người dùng chọn)/ 
- Để đăng nhập vào trang web với vai trò người dùng, thì ta nhập username: user1, password: 12345678
- Để đăng nhập vào trang web với vai trò admin thì ta nhập username:admin, password: 12345678. Sau đó trang sẽ dẫn admin đến một trang khác để xác thực một lần nữa, ta nhập Verify code là: 12345678
